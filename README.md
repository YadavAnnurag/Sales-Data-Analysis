# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data to identify trends, regional performance, product category performance, and sales representative effectiveness.

## Dataset
- 1000 sales records
- Includes sales amount, quantity sold, region, sales representative, customer type, payment method, and sales channel.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## KPIs
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value

## Analysis Performed
- Monthly Sales Trend
- Region-wise Analysis
- Product Category Analysis
- Sales Representative Performance
- Customer Type Analysis
- Sales Channel Analysis
